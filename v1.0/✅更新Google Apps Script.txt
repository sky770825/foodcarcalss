📋 問題：新增「尚未付款」資料可能被隱藏

✅ 解決：已修改 google_apps_script.js，自動顯示未付款行


═══════════════════════════════════════════════════════════════
            🚀 更新步驟（3步驟，1分鐘完成）
═══════════════════════════════════════════════════════════════

步驟1️⃣：打開 Google Apps Script
   ┌─────────────────────────────────────────────────┐
   │ 1. 打開您的 Google Sheets                       │
   │ 2. 點擊「擴充功能」→「Apps Script」             │
   │ 3. 會開啟腳本編輯器                             │
   └─────────────────────────────────────────────────┘

步驟2️⃣：複製新代碼
   ┌─────────────────────────────────────────────────┐
   │ 1. 打開這個文件：                               │
   │    📄 google_apps_script.js                     │
   │                                                  │
   │ 2. 全選代碼（Ctrl+A）                          │
   │ 3. 複製（Ctrl+C）                              │
   └─────────────────────────────────────────────────┘

步驟3️⃣：貼上並儲存
   ┌─────────────────────────────────────────────────┐
   │ 1. 在 Apps Script 編輯器中全選（Ctrl+A）       │
   │ 2. 貼上（Ctrl+V）                              │
   │ 3. 點擊💾儲存圖示（或按 Ctrl+S）              │
   │ 4. 完成！✨                                     │
   └─────────────────────────────────────────────────┘

⚠️ 不需要重新部署！儲存後立即生效。


═══════════════════════════════════════════════════════════════
                      ✨ 更新後的效果
═══════════════════════════════════════════════════════════════

✅ 新增「尚未付款」資料 → 一定會顯示
✅ 排序後「尚未付款」資料 → 自動顯示
✅ 已付款或己繳款資料 → 維持原本隱藏狀態
✅ 不影響其他隱藏的資料


═══════════════════════════════════════════════════════════════
                      🔍 如何驗證
═══════════════════════════════════════════════════════════════

測試方法：
  1. 在 Google Sheets 中隱藏一些「己繳款」的行
  2. 在網站上新增一筆預約（會是「尚未付款」）
  3. 檢查 Google Sheets
  
預期結果：
  ✅ 新增的「尚未付款」資料可以看到
  ✅ 原本隱藏的「己繳款」資料維持隱藏


═══════════════════════════════════════════════════════════════
                    💡 額外建議
═══════════════════════════════════════════════════════════════

更好的做法：使用「篩選器」代替「隱藏」

  優點：
    ✅ 更方便切換顯示/隱藏
    ✅ 不會與新資料衝突
    ✅ 可以儲存多個篩選條件
  
  設定方式：
    1. 選取標題行（第7行）
    2. 點擊「資料」→「建立篩選器」
    3. 點擊 H欄（款項結清）的下拉選單
    4. 勾選要顯示的項目（如：只顯示「尚未付款」）

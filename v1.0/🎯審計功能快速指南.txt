🎯 己繳場租審計功能 - 快速指南
=====================================

📌 功能說明
-----------
讓系統管理員可以將「尚未付款」的預約更新為「己繳款」狀態。

🔑 審計密碼
-----------
密碼：sky36990

📍 使用步驟
-----------

1. 在日曆上找到「尚未付款」的預約
   → 會看到綠色的 ✓ 審計按鈕（在事件左上角）

2. 點擊綠色審計按鈕
   → 打開「己繳場租審計」彈窗

3. 確認預約詳情無誤
   - 餐車名稱
   - 場地
   - 日期
   - 時間

4. 點擊「密碼確認繳費」按鈕

5. 輸入審計密碼：sky36990

6. 點擊「確認審計」

7. 系統自動更新 Google Sheets：
   - H欄：尚未付款 → 己繳款
   - I欄：添加「審計確認」標記

8. 完成！日曆會自動重新載入

✅ 顯示位置
-----------
- 審計按鈕（綠色圓形✓）：事件左上角
- 取消按鈕（紅色X）：事件右上角

⚠️ 重要提示
-----------
1. 此功能僅限系統管理員使用
2. 只有「尚未付款」的預約會顯示審計按鈕
3. 更新後無法從前端還原，請謹慎操作
4. 請勿將密碼洩漏給非管理員

📊 審計結果
-----------
更新成功後：
✓ Google Sheets H欄變更為「己繳款」
✓ 備註欄添加「審計確認」
✓ 日曆自動刷新顯示最新狀態
✓ F欄狀態自動更新為「己排班」

🔍 如何識別
-----------
- 尚未付款的預約：顯示倒數計時或「⏰ 繳費時間」
- 己繳款的預約：顯示「✓ 已付款」（綠色）
- 有審計按鈕：表示該預約尚未付款

💡 其他功能
-----------
如果只是要讓客戶繳費：
→ 點擊「⏰ 繳費時間」標籤
→ 會打開繳費彈窗（街口/LINE PAY/銀行轉帳）

如果要取消預約：
→ 點擊紅色 X 按鈕
→ 輸入取消密碼（同樣是 sky36990）

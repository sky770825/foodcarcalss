# 🔐 後台登入問題排查指南

## 預設密碼
**預設密碼：`admin123`**

## 常見問題

### 1. 密碼錯誤
- 確認輸入的密碼是：`admin123`
- 注意大小寫和空格

### 2. 登入按鈕無反應
檢查瀏覽器控制台（F12）是否有錯誤訊息

### 3. 頁面載入問題
- 清除瀏覽器快取：`Cmd+Shift+R` (Mac) 或 `Ctrl+Shift+R` (Windows)
- 檢查網路連線

## 快速修復

### 方法 1: 清除 Session Storage
在瀏覽器控制台執行：
```javascript
sessionStorage.clear();
location.reload();
```

### 方法 2: 直接修改密碼
如果忘記密碼，可以修改 `admin.js` 中的：
```javascript
const ADMIN_PASSWORD = 'admin123'; // 改成你想要的密碼
```

### 方法 3: 臨時移除登入驗證
如果需要臨時跳過登入（僅用於測試），可以在瀏覽器控制台執行：
```javascript
sessionStorage.setItem('adminLoggedIn', 'true');
location.reload();
```

## 調試步驟

1. **打開瀏覽器控制台**（F12）
2. **檢查是否有錯誤訊息**
3. **嘗試手動登入**：
   ```javascript
   // 在控制台執行
   document.getElementById('adminPassword').value = 'admin123';
   handleLogin();
   ```
4. **檢查元素是否存在**：
   ```javascript
   console.log('adminPassword:', document.getElementById('adminPassword'));
   console.log('loginError:', document.getElementById('loginError'));
   console.log('loginModal:', document.getElementById('loginModal'));
   ```

## 如果仍然無法登入

請提供以下資訊：
1. 瀏覽器控制台的錯誤訊息
2. 輸入的密碼（確認是否正確）
3. 是否有看到登入畫面
4. 點擊登入按鈕後是否有任何反應
